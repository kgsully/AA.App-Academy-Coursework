Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one (and only one) 'h1'.

Paragraphs

Paragraphs of text go in between 'p' tags.

Lists

There are 2 types of lists - Unordered and Ordered. Note that lists can be nested.

Unordered Lists

Unoredered lists are marked with bullet points and go in 'ul' tags. Each list item goes in an 'li' tag.

Types of lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

Ordered lists are numbered and go in 'ol' tags. Each list item goes in an 'li' tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to DuckDuckGo to search for more information.

Setting 'target="_balnk"' will open the link in a new window or tab. See MDN for more details.

Text Emphasis

Italic

The 'em' element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the 'strong' element to emphasize text. This shows up as bolt text in most browsers.
