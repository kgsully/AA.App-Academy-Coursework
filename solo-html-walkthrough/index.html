My HTML Cheat Sheet

Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one and ONLY one 'h1'.

Paragraphs

paragraphs of long-form text containing complete sentences go in 'p' tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in 'ul' tags. Each list item goes in an 'li' tag.

Types of Lists:

  • Unordered Lists ('ul')
  • Ordered Lists ('ol')

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. According to "HTML for Dummies" - 'An anchor element is called an anchor because web designers can use it to "anchor" a URL to some text on a web page.'

Go to DuckDuckGo to search for more information.

Setting `target="_blank"` will open the link in a new window. See MDN for more details.

Images

You can use the img tag to add images to your page. `src="PATH TO IMAGE"` sets the image location and `alt=""` will provide a text description should the image fail to load (or for a screen reader to interpret to aid visually impared users).

Tables

Tables in HTML are great for displaying tabluar data, such as an address book or a list of product descriptions / prices. Although tables are good for keeping things organized, they should only be used for displaying data, not for defining page layout.

Create the table using the 'table' tag.
If the table has a header (that is, rows that should be considered as the column headers), you add a 'thead' tag.
Then you add a 'tbody' tag which should contain the data of your table.
If your table has a footer (that is, rows that summarize the tabular data) then you ad a 'tfoot' also.

Rows are added between the table "sections" using the table row ('tr') tag.
In the rows, put 'th' elements and signify that it is a column or row header using the "scope" attribute.
Use 'td' tags to hold the data.
